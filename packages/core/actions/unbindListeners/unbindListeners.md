# UnbindListeners

Unbind Keybind library.

## Parameters

| Name   | Type        | Mandatory | Default |
| ------ | ----------- | --------- | ------- |
| target | HTMLElement | true      | -       |

## Returns

void

## Usage

```ts
import { unbindListeners } from "keybind";

// Init KeyBind and do some stuff ...

unbindListeners(document.body);
```
